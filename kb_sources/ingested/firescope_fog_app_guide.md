FIRESCOPE Mobile App User Guide Page | 1 1 Introduction 1.1 What is FIRESCOPE mobile app? FIRESCOPE p rovid es crucial information to first responders since 1972, FIRESCOPE is proud to present the interactive 2022 Field Operations Guide ICS 420 - 1. All the information from the 2022 printed FOG ICS 420 - 1 is included in this app. Now you can easily find critical in formation on Resource Typing, ICS Forms, Position Checklists, Organization Charts and examples of how to organize an incident using the Incident Command System (ICS) when facing All - Hazard Incidents. New features to this update include the ability to vie w in landscape or portrait view. Zoom is now functional and allows for easier reading. ICS forms can be viewed and filled out using Adobe Acrobat. Traditional chapter viewing is also available. Easily access other valuable information such as Wildland Fire Management Guiding Principles, Leaders Intent, Tactical Engagement, Levels of Engagement, Powerline Safety and a Structure Assessment Checklist. Also included is valuable information on these ICS topics: 1. Responder Rehab 2. Common Responsibilities 3. Multi - Agency Coordination System (MACS) 4. Area Command 5. Complex 6. Command 7. Unified Command 8. Planning Process 9. Operations 10. Planning 11. Logistics 12. Finance/Administration 13. Organizational Guides 14. Resource Types and Minimum Standards 15. Hazardous Materials 16. Multi - Casualty Incidents 17. Urban Search and Rescue 18. Swiftwater/Flood Search and Rescue
